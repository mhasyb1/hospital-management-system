## 📋 Project Overview

A complete Hospital Management System to digitally manage hospital operations:

- **👥 Patient Management** — Registration, profiles, medical history
- **👨‍⚕️ Doctor Management** — Specializations, schedules, availability
- **📅 Appointment System** — Booking, scheduling, confirmations, reminders
- **💳 Billing System** — Invoice generation, payment tracking
- **📊 Admin Dashboard** — Reports, analytics, user management

### Technical Stack

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js with Express.js
- **Database:** MongoDB with Mongoose ODM
- **Version Control:** Git & GitHub
- **Project Management:** GitHub Projects
